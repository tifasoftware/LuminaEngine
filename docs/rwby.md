# Project: RWBY
*Last Updated 6/10/2026*

## What is Project RWBY
Project RWBY is a series of porting experiments where the Lumina Engine is ported to four separate consoles from the sixth generation.
The project is named after a titular character group, Team RWBY, which (like the sixth-generation console) represents four members.

In these series of experiements, the engine will be ported to these four consoles:
- Sega Dreamcast
- Sony PlayStation
- Nintendo GameCube
- Microsoft Xbox (OG)

In addition, the project will also be documented in a formal research paper, followed by a possible YouTube video adaptation.
